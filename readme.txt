Manage a band directory, creating artists, albums and songs, as a band/artist discography listing.

== Description ==

WP Bands Directory is a plugin that creates custom post types for artists, albums, and songs so that they can be displayed in a directory fashion.

== Installation ==

1. Upload 'wp-bands-directory' folder to the '/content/plugins/' directory or use the 'Add New' option under the 'Plugins' menu in your WordPress backend and 'Upload' 'wp-bands-directory.zip'.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure settings for WP Bands Directory in the WordPress dashboard by clicking on the 'Bands Directory' option in your WordPress dashboard.
4. Enter the appropriate information into the form fields in the 'WP Bands Directory URL Rewrites' section of the 'WP Bands Directory General Settings' page.
5. Copy the artist/album/song template files to your theme's directory by clicking on the checkboxes in the 'WP Bands Directory Template Files' section of the 'WP Bands Directory General Settings' page.
6. Create your Artists/Albums/Songs listings by using the appropriate tabs in your WP Dashboard.
7. Create a page for your "directory" (page that shows all artists and links to the artist pages) by pasting the [wbd_artists] shortcode into the editor for that page.
